<br/>

<h3>Features</h3>

<h6>Event List Page: </h6>
<p>
<ul>
<li><p>Browse a comprehensive list of upcoming events.</p></li>
<li><p>Sort events by name, date, or organizer.</p></li>
<li><p>Click on an event to view details.</p></li>
</ul>
 
<h6>Event Details Page: </h6>
<ul>
<li><p>Explore event-specific information.</p></li>
<li><p>View the list of participants.</p></li>
<li><p>Filter participants by full name or email.</p></li>
</ul>

<h6>Participant Registration Page: </h6>
<ul>
<li><p>Register as a participant for your favorite events.</p></li>
<li><p>Fill out the registration form with data validation using Yap.</p></li>
<li><p>Upon successful submission, navigate to the selected event page.</p></li>
</ul>

<h5>Getting Started: </h5>
<ol>
<li><p>Clone this repository.</p></li>
<li><p>Install dependencies: npm install.</p></li>
<li><p>Start the development server: npm start.</p></li>
<li><p>Open your browser and visit http://localhost:3000.</p></li>
</ol>
